# Data Warehouse e Data Lake

## O que o código faz?

Primeiro ele cria e manipula conjuntos de dados de produtos e vendas, salva-os em arquivos CSV, lê-os de volta, mescla eles e imprime o resultado.

Depois ele realiza uma análise de vendas detalhada de um conjunto de dados de vendas e produtos, ele lê os dados de um arquivo CSV, realiza várias análises agrupadas (por produto, categoria e data), cria um gráfico de vendas ao longo do tempo e imprime os resultados, as análises incluem a soma da quantidade e do valor total de vendas para cada agrupamento.

Depois ele cria um diretório chamado ‘data_lake’ se ele não existir, gera 10 arquivos CSV com 1000 linhas de dados aleatórios cada, salva esses arquivos no diretório ‘data_lake’, e imprime as primeiras cinco linhas de cada arquivo.

Depois ele importa esses dados para o banco dados e depois de importado ele gera mais relatórios e também três gráficos
